## Tagline Guidelines

### When to Use Which

1. **Core brand tagline** — Always use for company-wide materials
2. **Division taglines** — Use for division-specific materials
3. **Feature taglines** — Use for feature-specific marketing
4. **Use case taglines** — Use for audience-specific materials

### Tone

- **Technical** — For developer/researcher audiences
- **Aspirational** — For marketing/investor audiences
- **Descriptive** — For documentation/product pages

### Consistency

- Always include "ekkOS" in taglines
- Maintain the "cognitive operating system" positioning
- Emphasize "memory," "learning," "intelligence" themes
